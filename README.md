# Wind-tunnel
3D printable mini wind tunnel with Raspberry Pi 5 for Hack Club Blueprint
Thank you to everyone from Hack Club for hosting this event!

Things neede to build this:

Toggle switches that are soderable
Rotary encoder with push button
Micro HDMI to HDMI cable
USB male to female extender
Liquid electrical tape*
CSI FPC extension cable
1/16" thick acrylic sheets, at least 4X6, and need 3
30X30X7mm fan for Raspberry Pi
Camera for compatible with Raspberry Pi 5 with CSI
Heat shrink tubing*
GPIO extension board for Raspberry Pi 5
Cooler for Raspberry Pi 5*
Raspberry Pi 5 8GB
Mini/DIY mist maker/humidifier
Ultra thin LED strip (I recommend 2.7mm)*
Electronic Wire (Soderable)
120mm PC Fan
7" HDMI screen (Make sure that it does not have anything else built in)

*Not required, but highly reccomended

Sodering Iron and 3D Printer required
